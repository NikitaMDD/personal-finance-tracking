# Design Tokens

## Назначение

Для обеспечения единообразия пользовательского интерфейса была разработана собственная система дизайн-токенов.

Дизайн-токены представляют собой набор базовых значений, описывающих визуальный стиль приложения. Использование токенов позволяет централизованно управлять внешним видом интерфейса, обеспечивать единообразие компонентов и значительно упрощать поддержку проекта.

В качестве механизма хранения токенов используются CSS Custom Properties.

# Типографика

Основным шрифтом приложения выбран **Inter**.

Причины выбора:

- высокая читаемость;
- современный минималистичный стиль;
- широкое распространение в современных веб-приложениях.

Размеры текста:

| Стиль | Размер |
|--------|----------|
| Display | 40 px |
| H1 | 32 px |
| H2 | 24 px |
| H3 | 20 px |
| Body | 16 px |
| Small | 14 px |
| Caption | 12 px |

# Анимации

Все переходы интерфейса имеют продолжительность от 150 до 200 мс.

Используются следующие свойства:

- opacity;
- transform;
- background-color.

Использование сложных анимаций исключено в целях повышения скорости восприятия интерфейса.

---

# Принципы проектирования

При разработке дизайн-системы были приняты следующие принципы:

- минимализм;
- единообразие компонентов;
- переиспользуемость;
- адаптивность;
- масштабируемость.
