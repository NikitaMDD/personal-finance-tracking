# Design Guidelines

## Назначение

Настоящий документ определяет правила использования дизайн-токенов и компонентов пользовательского интерфейса.

Главная цель документа — обеспечить единообразие интерфейса и исключить появление несогласованных визуальных решений.

---

# Общие принципы

При разработке интерфейса используются следующие принципы.

- единая дизайн-система;
- минимализм;
- переиспользование компонентов;
- отсутствие произвольных размеров;
- отсутствие произвольных цветов;
- отсутствие произвольных анимаций.

---

# Использование цветов

Все цвета интерфейса должны использоваться только через дизайн-токены.

Разрешено:

```css
background: var(--color-primary);
```

Не допускается:

```css
background: #2563eb;
```

---

# Использование отступов

Все внешние и внутренние отступы должны использовать значения из spacing-токенов.

Разрешено:

```css
padding: var(--space-4);
```

Запрещается:

```css
padding: 13px;
```

---

# Использование радиусов

Для всех компонентов используются только значения из radius.css.

Не допускается использование произвольных скруглений.

---

# Использование теней

Тени применяются только:

- карточкам;
- модальным окнам;
- выпадающим спискам.

Использование теней для кнопок и полей ввода не рекомендуется.

---

# Использование анимаций

Продолжительность всех анимаций должна использовать значения transition-токенов.

Разрешено:

```css
transition: var(--transition-fast);
```

Не допускается:

```css
transition: .37s ease;
```

---

# Использование типографики

Все размеры текста должны использовать переменные typography.css.

Разрешено:

```css
font-size: var(--font-body);
```

Запрещается:

```css
font-size: 17px;
```

---

# Использование компонентов

Каждый новый экран должен собираться исключительно из компонентов UI Kit.

Создание локальных кнопок, полей ввода или карточек внутри страниц не допускается.

---

# Основные принципы

Любой элемент интерфейса должен отвечать следующим требованиям:

- переиспользуемость;
- адаптивность;
- доступность;
- типизация;
- независимость от конкретной страницы.
